第10回課題

Webサイトフレームワークのレファレンスサイトを読み、index.htmlをデザインする。

レファレンスサイトを読んでわかったこと

UIkit

  • ダウンロードすると、CSSファイルとJavaScriptファイルが入ったzipファイルを展開することができる。
  • インストールとコンパイルが完了したら、対象のhtmlファイルのコードのheadに以下を記述する。UIkit icon libraryも含まれている。基本的なセットアップはこれで完了。
    •               <head>
                <title>Title</title>
                <meta charset="utf-8">
                <meta name="viewport" content="width=device-width, initial-scale=1">
                <link rel="stylesheet" href="css/uikit.min.css" />
                <script src="js/uikit.min.js"></script>
                <script src="js/uikit-icons.min.js"></script>
            </head>
  • UIkitは作業効率の観点から、AtomやSublime Textなどのsolid code editorでの使用が推奨されている。
  • UIkitはあらゆるモダンブラウザにおいて正常な動作が可能である。

Bootstrap

  • UIkitと同様にCSSファイルとJavaScriptファイルがダウンロードできる。
  • インストールとコンパイルが完了したら、対象のhtmlファイルのコードのheadに以下を記述する。
    •               <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css" integrity="sha384-Gn5384xqQ1aoWXA+058RXPxPg6fy4IWvTNh0E263XmFcJlSAwiGgFAW/dAiS6JXm" crossorigin="anonymous">
            
                  <script src="https://code.jquery.com/jquery-3.2.1.slim.min.js" integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" crossorigin="anonymous"></script>
                <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js" integrity="sha384-ApNbgh9B+Y1QKtv3Rn7W3mgPxhU9K/ScQsAP7hUibX39j7fakFPskvXusvfa0b4Q" crossorigin="anonymous"></script>
                <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js" integrity="sha384-JZR6Spejh4U02d8jOt6vLEHfe/JQGiRRSQQxSfFWpi1MquVdAyjUar5+76PVCmYl" crossorigin="anonymous"></script>
  • また、上記のコードを含んだhtml全体のテンプレートも公開されている。
    •               <!doctype html>
                <html lang="en">
                  <head>
                    <!-- Required meta tags -->
                    <meta charset="utf-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
                
                    <!-- Bootstrap CSS -->
                    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css" integrity="sha384-Gn5384xqQ1aoWXA+058RXPxPg6fy4IWvTNh0E263XmFcJlSAwiGgFAW/dAiS6JXm" crossorigin="anonymous">
                
                    <title>Hello, world!</title>
                  </head>
                  <body>
                    <h1>Hello, world!</h1>
                
                    <!-- Optional JavaScript -->
                    <!-- jQuery first, then Popper.js, then Bootstrap JS -->
                    <script src="https://code.jquery.com/jquery-3.2.1.slim.min.js" integrity="sha384-KJ3o2DKtIkvYIK3UENzmM7KCkRr/rE9/Qpg6aAZGJwFDMVNA/GpGFF93hXpG5KkN" crossorigin="anonymous"></script>
                    <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js" integrity="sha384-ApNbgh9B+Y1QKtv3Rn7W3mgPxhU9K/ScQsAP7hUibX39j7fakFPskvXusvfa0b4Q" crossorigin="anonymous"></script>
                    <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js" integrity="sha384-JZR6Spejh4U02d8jOt6vLEHfe/JQGiRRSQQxSfFWpi1MquVdAyjUar5+76PVCmYl" crossorigin="anonymous"></script>
                  </body>
                </html>
  • Responsive meta tagの使用も推奨されている。
    •               <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
  • Google MapsやGoogle Custom Search Engineでのbox-sizingに関する不具合は、以下のコードを記述することで回避する。
    •               .selector-for-some-widget {
                box-sizing: content-box;
              }

vue,js

  • Vueは増加的に適応化するようにデザインされたprogressive frameworkである。
  • core libraryはview　layerのみに集中しているため、他のlibraryや既存プロジェクトとの統合が簡単である。
  • modern toolingやsupporting librariesとの併用でのSingle-Page Applicationsのpoweringに非常に適している。
  • インストールとコンパイルが完了したら、対象のhtmlファイルのコードのheadに以下のいずれかのコードを記述する。
    •               <script src="https://cdn.jsdelivr.net/npm/vue@2/dist/vue.js"></script>
            
                  <script src="https://cdn.jsdelivr.net/npm/vue@2"></script>
  • Declarative Rendering、Conditionals and Loops、Handling User Input、Composing with ComponentsなどについてのHTMLとCSSサンプルコードが公開されている。

index.htmlのデザイン

  • gridのデザイン
    • ContentsとAssignmentsのgridの設定を書き換えて、レイアウトを変更する。
      •                  <div class="uk-child-width-1-3@s uk-child-width-1-3@m uk-grid-small" uk-grid>
    • "...-1-3..."は縦に3分割することを指定していることが分かる。
  • cardのデザイン
    • ContentsとAssignmentsのcardを書き換えて、カラーを変更する。
      •                 <div class="uk-card uk-card-hover uk-card-primary uk-card-body">
              
                      <div class="uk-card uk-card-hover uk-card-secondary uk-card-body">
    • レスポンシブデザインであることも確認できる。
    • UIkitにおいてはcardのカラーは"Default"(白)、"Primary"(青)、"Secondary"(黒)から指定できる。
