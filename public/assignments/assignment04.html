第４回課題

ドキュメンテーションのテンプレートHTML

aboutme.htmlの作成

  • "New file"をクリックして、publicフォルダ内に、"aboutme.html"ファイルを作る。
  • 最初にDOCTYPE宣言をする。※補完機能を利用して効率的に書けるように慣れる。
    •         <!DOCTYPE html>
  • index.html内のheadの内容をaboutme.htmlにコピペする。
    •          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1">
            <title>Welcome to Ibuki Shibayama's Website</title>
            <link rel="stylesheet" href="./css/uikit.min.css">
            <script src="./js/uikit.min.js"></script>
            <script src="./js/uikit-icons.min.js"></script>
            </script>
            <script src="https://code.jquery.com/jquery-3.5.1.min.js"></script>
            <script type="text/javascript">
            
              $(function(){
                $("#Header").load("./nav.html");
              });
        
            </script>
        </head>
  • bodyタグの中に見出しのhタグやパラグラフのpタグを作り、自己紹介のテキストを挿入する。※リストタグやリンクを貼ることができるaタグを上手く利用する。
    •           <body>
            </header>
            <h1>自己紹介</h1>
            <p>柴山息吹</p>
            <ul>
                <li>所属</li>
                    <ul>
                        <li>青山学院大学</li>
                            <ul>
                                <li><a href="https://www.aoyama.ac.jp/">青山学院大学ホームページ</a></li>
                            </ul>
                        <li>地球社会共生学部</li>
                            <ul>
                                <li><a href="https://www.gsc.aoyama.ac.jp/">地球社会共生学部ホームページ</a></li>
                            </ul>
                        <li>古橋研究室</li>
                            <ul>
                                <li><a href="http://furuhashilab.com/">古橋研究室ホームページ</a></li>
                                <li><a href="https://github.com/furuhashilab">Furuhashi Labs Github</a></li>
                                <li><a href="https://goo.gl/maps/hTbB1j2rmz65THE6A">Google map</a></li>
                            </ul>
                    </ul>
                    
    
                <li>出身：神奈川県相模原市</li>
                <li>趣味</li>
                    <ul>
                        <li>アニメ・映画</li>
                        <li>プラモデル</li>
                        <li>ラーメン</li>
                    </ul>
            </ul>
        </body>
  • "New Folder"をクリックして、publicフォルダ内に、"img"フォルダーを作る。
  • 挿入する写真を選択し、ファイル名が半角英数字スペース無しであることを確認する。そして、ファイルを右クリックして"編集"を開き、"サイズ変更"でピクセル数を調整し、200KB以下までサイズダウンさせる。
  • サイズダウンした写真はimgフォルダにコピペする。
  • imgタグを使って写真ファイルを挿入する。属性はdata-srcで、属性値には相対パスを使ってファイル名を入力する。
    •                 <img data-src="./img/okinawa3small.jpg" width="50%" uk-img>

HOMEからABOUTMEへアクセスできるようにする

  • nav.htmlのAboutMeのaタグのファイル名を./aboutme.htmlにする。
    •                    <li class="uk-active"><a href="./aboutme.html">AboutMe</a></li>
