第8回課題

スタイルシートを試す。

test-style.cssの作成

  • cssフォルダを開き、"New File"をクリックして"test-style.css"ファイルを作る。
  • 授業用webページを見ながら一からコードを書き写す。
    •                     body {
                    margin: 0;
                    text-align: center;
                }
                
                h2 {
                        font-size: 20px;
                        color: blueviolet;
                }
                #media {
                        background-color: aqua;
                }
                .image {
                    margin-left: auto;
                    margin-right: auto;
                    width:70%;
                    padding: 10px;
                }
  • 保存する。

csstest.htmlの作成

  • publicフォルダを開き、"New File"をクリックして"csstest.html"ファイルを作る。
  • 授業用webページを見ながら一からコードを書き写す。
    •                     <html lang="ja">
                    <head>
                        <title>CSS Sample Test</title>
                        <meta charaset="utf-8">
                        <limk rel="stylesheet" href="./css/test-style.css">
                    </head>
                    <body>
                        <h2>CSS Sample Test</h2>
                        <div id="media">
                            <img class="image" src="./img/okinawa2.jpg" width="100%">
                            <img class="image" src="./img/okinawa3small.jpg" width="100%">
                            <img class="image" src="./img/sample.jpg" width="100%"> 
                        </div>
                    </body>
                </html>
  • 保存し、Git Bashを使ってWebにコミットさせる。

Webでの動作の確認

  • 最初に確認すると、cssが丸ごと指定できてない状態だったため、コードを確認すると、csstest.htmlのlinkタグに誤字(link→limk)があった。
    •                 <limk rel="stylesheet" href="./css/test-style.css">
  • コードを修正し、再度Webを確認すると、狙い通りにコミットされていることが確認できた。
  • ブラウザのウィンドウを狭めると、余白や間隔が維持されながら画像も縮小されていた。

responsive.cssの適用

  • csstest.htmlからtest-style.cssを削除する。
  • publicフォルダ内に"responsive.html"ファイルを新規作成し、授業用Webページを見ながらmedia queriesのサンプルコードを書き写す。
    •                     <html lang="ja">
                    <head>
                        <title>Responsive Design Test</title>
                        <meta charaset="UTF=8">
                        <meta name="viewport" content="width=device-width, user-scalable=no, maximum-scale=1, initial-scale=1" />
                        <link rel="stylesheet" href="./css/responsive.css">
                    </head>
                    <body>
                        <div id="content">
                            <h1>Responsive Design Test</h1>
                            <div class="imgblock">
                                <img class="image" src="./img/okinawa2.jpg">
                                <img class="image" src="./img/okinawa3small.jpg">
                                <img class="image" src="./img/sample.jpg">
                            </div>
                        </div>
                    </body>
                </html>
  • CSSフォルダ内に"responsive.css"ファイルを新規作成し、授業用Webページを見ながらスタイルシートを書き写す。
    •                 h1 {
                    color: #FF0000;
                    text-align: center;
                }
                #content {
                    width: 920px;
                    background-color: #CCCCCC;
                    margin: 0 auto;
                }
                .imgblock {
                    width: 900px;
                    text-align: center;
                }
                .image {
                    width: 30%;
                    float: none;
                }
                
                @media screen and (max-width: 1024px){
                
                    h1 {
                        color: #00AA00;
                        text-align: center;
                    }
                    #content {
                        width: 780px;
                        margin: 0 auto;
                    }
                    .imgblock {
                        margin: 0 auto;
                        width: 750px;
                        text-align: center;
                    }
                    .image {
                        width: 40%;
                        float: none;
                    }
                }
                
                @media screen and (max-width: 768px) {
                
                    h1 {
                        color: #0000FF;
                        text-align: center;
                    }
                    #content {
                        width: 100%;
                    }
                    .imgblock {
                        width: 100%;
                        text-align: center;
                    }
                    .image {
                        width: 100%;
                        margin: 0 auto;
                    }
                }
  • サーバーにアップロードする。

PCとスマホのブラウザでの見え方の確認

  • 1.PCのブラウザでの見え方
  • ウィンドウを小さくすると、中くらいのサイズでh1のテキストが緑色に、一番小さくすると青色に変わった。また、それに伴って画像の並びも横2列、縦1列と変化した。
  • 2.スマホでの見え方
  • 動作に問題は無かったため、csstest.htmlにもこのスタイルシートを適用させる。
